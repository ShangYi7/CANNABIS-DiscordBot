### 要求
- Discord Server
- node
- JavaScript
- 腦袋

###  主要特點
- JavaScript V14
- Slash Commands “/” 
- 基本指令：
   - ping 
   - ban (禁止成員)
   - unban (解除用戶靜止)
   - clear (刪除訊息)
   - kick (剔除成員)
   - mute (靜言)
   - giveaway (抽獎)
- 加入日誌 (當我玩家進到伺服器時)
- 離開日誌 (當玩家與離開伺服器)
- 驗證 (/verify)
- Ticket (票口)
